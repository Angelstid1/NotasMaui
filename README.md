# NotasMaui - Aplicación de Notas con .NET MAUI

## Descripción
Aplicación móvil de notas desarrollada con .NET MAUI que permite crear, editar, eliminar y buscar notas con persistencia en SQLite.

## Requisitos de Compilación

### Entorno de Desarrollo
- Visual Studio 2022** (v17.8+) con workload de .NET MAUI
- .NET SDK 8.0+
- Android SDK (API 24+)
- Git para control de versiones

### Versiones Compatibles
- Android: 5.0 (API 21) o superior
- iOS: 11.0 o superior (si aplica)
- Windows: 11/10 (opcional)

### Clonar el repositorio
git clone https://github.com/Angelstid1/NotasMaui.git
cd NotasMaui
